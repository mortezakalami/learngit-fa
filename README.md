# گیت چیست؟

**گیت (Git)** یک سیستم کنترل نسخهٔ توزیع‌شده (Distributed Version Control System) است که ابزار اصلی توسعه‌دهندگان نرم‌افزار برای مدیریت، ردیابی و بازگردانی تغییرات کد محسوب می‌شود.

هر بار که تغییری در پروژه اعمال می‌کنید — مانند رفع باگ، افزودن قابلیت جدید یا بازنویسی بخشی از کد — گیت وضعیت کامل پروژه را در آن لحظه به‌صورت یک نسخه (**Commit**) ذخیره می‌کند.

این امکان را فراهم می‌کند که:

- به نسخه‌های قبلی بازگردید  
- تاریخچهٔ تغییرات را ببینید  
- توسعهٔ هم‌زمان داشته باشید  
- تغییرات را بدون تداخل ادغام (**Merge**) کنید  

---

## مخزن (Repository) چیست؟

**مخزن (Repository)** پوشه‌ای است که گیت آن را مدیریت می‌کند.

درون آن وجود دارد:

- فایل‌ها  
- commitها  
- شاخه‌ها  
- تنظیمات گیت  

به‌صورت محلی یا remote ذخیره می‌شود.

---

## گیت‌هاب (GitHub) چیست؟
**گیت‌هاب**، پلتفرم مبتنی بر وب و محبوب‌ترین سرویس میزبانی مخازن گیت در جهان است که توسط مایکروسافت اداره می‌شود. 
گیت تغییرات کد را به‌صورت محلی مدیریت و ذخیره می‌کند، اما گیت‌هاب این مخازن را در فضای ابری (cloud) میزبانی می‌نماید و امکانات پیشرفته‌ای برای همکاری تیمی فراهم می‌آورد؛ از جمله: - اشتراک‌گذاری کد با دیگران - ایجاد **pull request** برای بررسی و ادغام تغییرات - پیگیری مسائل (**issues**) - بررسی کد (**code review**) - مدیریت پروژه با ابزارهای یکپارچه - میزبانی وب‌سایت‌های استاتیک (**GitHub Pages**)
